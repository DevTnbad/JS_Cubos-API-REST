# Back-end | API REST

### Pré-requisitos:
- Node.js

### Instalação:
1. Clone este repositório:
**git clone https://github.com/seu-usuario/nome-do-projeto.git**

2. Navegue até o diretório do projeto: 
**cd/numero/nome-do-projeto**

3. instale as dependências:
**npm install**

# Back-end |3 API's REST

### Pré-requisitos:
- Node.js

### Instalação:
1. Clone este repositório:
```
**git clone https://github.com/DevTnbad/JS_Cubos-API-REST**
```


2. Navegue até o diretório do projeto: 
```
**cd JS_Cubos-API-REST**
```


3. instale as dependências:
```
**npm install**
```
